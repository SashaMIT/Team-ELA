        <h2>Detailed Calculation Explanation</h2>
        <p>This calculation is based on the value of Bitcoin's mining security and Elastos' share of that security through merged mining. Here's a step-by-step breakdown:</p>
        <ol>
            <li><strong>Bitcoin Mining Rewards:</strong>
                <ul>
                    <li>Block Reward: 3.125 BTC (every 10 minutes)</li>
                    <li>Blocks per Year: 52,560 (6 blocks per hour * 24 hours * 365 days)</li>
                    <li>Annual Mining Rewards (BTC): 164,250 BTC/year (3.125 * 52,560)</li>
                </ul>
            </li>
            <li><strong>Annual Value of Bitcoin's Hashpower:</strong>
                <ul>
                    <li>Annual Mining Rewards (USD) = Annual Mining Rewards (BTC) * Bitcoin Price</li>
                    <li>Example: 164,250 BTC * $63,796 = $10.48 billion/year</li>
                </ul>
            </li>
            <li><strong>Elastos' Hashpower Share:</strong>
                <ul>
                    <li>Elastos' Share of Hashrate: 45%</li>
                    <li>Annual Value of Elastos' Hashpower = Annual Value of Bitcoin's Hashpower * 45%</li>
                    <li>Example: $10.48 billion * 45% = $4.716 billion</li>
                </ul>
            </li>
            <li><strong>ELA Token Value Calculation:</strong>
                <ul>
                    <li>Total Supply of ELA: 26,220,000 tokens</li>
                    <li>Per ELA Token Value = Annual Value of Elastos' Hashpower / Total Supply of ELA</li>
                    <li>Example: $4.716 billion / 26,220,000 = $179.86 per ELA token</li>
                </ul>
            </li>
        </ol>
        <p>This calculation assumes that the value of ELA is directly proportional to its share of Bitcoin's mining rewards, based on its 45% hashrate contribution through merged mining. It's important to note that this is a simplified model focusing solely on the security aspect and does not account for other factors that could influence ELA's value.</p>

    <script>
        const bitcoinPriceSlider = document.getElementById('bitcoin-price');
        const bitcoinPriceValue = document.getElementById('bitcoin-price-value');
        const bitcoinHashrateInput = document.getElementById('bitcoin-hashrate');
        const elastosHashrateInput = document.getElementById('elastos-hashrate');
        const marketPriceInput = document.getElementById('market-price');
        const estimatedValueDiv = document.getElementById('estimated-value');
        const potentialUpsideSpan = document.getElementById('potential-upside');

        const elaSupply = 26.22; // Updated ELA supply in millions

        function calculateELAValue() {
            const bitcoinPrice = parseFloat(bitcoinPriceSlider.value);
            const elastosHashrate = parseFloat(elastosHashrateInput.value);
            const btcBlockReward = 3.125;
            const blocksPerYear = 52560;
            const annualBTCRewards = btcBlockReward * blocksPerYear;
            const annualUSDRewards = annualBTCRewards * bitcoinPrice;
            const elastosHashpowerValue = (annualUSDRewards * elastosHashrate) / 100;
            return elastosHashpowerValue / (elaSupply * 1000000);
        }

        function updateCalculation() {
            const bitcoinPrice = parseFloat(bitcoinPriceSlider.value);
            bitcoinPriceValue.textContent = bitcoinPrice;

            const elaValue = calculateELAValue();
            estimatedValueDiv.textContent = `Estimated ELA Value: $${elaValue.toFixed(2)}`;

            const currentMarketPrice = parseFloat(marketPriceInput.value);
            const potentialUpside = ((elaValue / currentMarketPrice - 1) * 100).toFixed(2);
            potentialUpsideSpan.textContent = potentialUpside;
        }

        bitcoinPriceSlider.addEventListener('input', updateCalculation);
        bitcoinHashrateInput.addEventListener('input', updateCalculation);
        elastosHashrateInput.addEventListener('input', updateCalculation);
        marketPriceInput.addEventListener('input', updateCalculation);

        updateCalculation(); // Initial calculation
    </script>
